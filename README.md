# Monitoring Pasien

## How to Run

```
python3 -m venv venv
source venv/activate/bin
pip3 install -r requirements.txt
```

### Terminal 1
```
source venv/activate/bin
python backend/app.py
```


### Terminal 2
```
source venv/activate/bin
python dashboard/app.py
```

### Terminal 3
```
source venv/activate/bin
python script/capture.py
```
